Bad Apple C#

One of the bigger projects I've been putting time in recently, is my rendition of the music video for Bad Apple in a C# console. While I could have quite easily imported a library that can play MP4 videos and be done, I decided that like many other C# renditions, it should run with ascii art. But since I couldn't easily convert the MP4 to images to ascii, I decided to lend the code from another person (Linked in README). The project is a good way of learning many functions and threading, now it's moving on to be a good way to learn music theory and how to use libraries like NAudio. Hopefully I'll be able to finish it someday.
